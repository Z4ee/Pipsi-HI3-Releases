<h1 align="center">Getting Started</h1>

### Usage

1. Download last release.
2. Run `Launcher.exe`.
3. Select `BH3.exe`.
4. After opening the game, press `INSERT` to open the GUI.

<h1 align="center">Features</h1>

#### Player
- Kill Aura
- God Mode
- Attack Speed
- Move Speed
- Attack Multiplier
- Noclip

#### World

- Instant Win
- Dump Enemies
- Auto Skip Dialogs

#### Visuals

- Peeking
- Fov Changer

#### Misc

- Skin Changer
- Auto Challenge
- Unload
- FPS Indicator
- Load/Save/Reset Config
